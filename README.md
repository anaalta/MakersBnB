User stories

Sandy and Daniel workingon the Log in and Out Feature


```
"As a user
So I can use AirBnB
I would like to be able to sign up to AirBnB"

"As a signed-up user
So I can close my session
I would like to be able to log out of AirBnB"

"As a logged-out user
So I can use the app at a second moment
I would like to be able to log in"

"As a property owner
So I can offer accommodation to customers
I would like to be able to list a place"

"As a property owner
So I can give specifications about my place
I would like to be able to name my space, describe it and set a price per night"

"As a propery owner
So I can maximise rental of my portfolio
I would like to be able to list multiple spaces"

"As a property owner
So I can rent my space flexibly
I would like to be able to select a range of dates when the property is available"

"As a sign-up user,
So I can rent a space,
I want to be able to hire any available space for the night."

"As a property owner,
So I can easily manage my requests
I'll ike to be able to aprrove, or decline, my booking requests"

"As a property owner,
So that I won't get overlaps in bookings,
I want my listing not to be available to other users on dates when it's booked"

"As a property owner
So I can maximise bookings
I would like my space to be available until a booking request is confirmed for that date"

"As a user,
So I can see my booking on the go,
I want to be able to get a confirmation by sms."
```
